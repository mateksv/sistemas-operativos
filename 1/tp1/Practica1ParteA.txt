6) El directorio raíz “/”es la base para todos los directorios en Linux.
• /bin: se encuentran los programas esenciales para el arranque del sistema y su funcionamiento.
• /boot: contiene el núcleo de Linux (kernel), la imagen inicial de la RAM y el bootloader (gestor de
arranque).
• /dev: contiene archivos de todos los dispositivos que Linux puede reconocer.
• /etc: contiene los archivos de configuración de todo el sistema y scripts Shell que se ejecutan al
arranque del sistema
• /home: es donde se encuentran una carpeta por cada usuario. El nombre de cada subcarpeta
corresponde al nombre de cada usuario.
• /lib: contiene librerías requeridas por los programas que están en /bin.
• /lost+found: es una carpeta usada para la recuperación de datos en caso de corrupción de
archivos
• /media: es donde se montan automáticamente dispositivos como memorias USB, CD, etc.
• /mnt: similar a /media, pero los dispositivos se montan manualmente.
• /opt: usualmente es usada para instalar programas comerciales (de terceros) en el sistema.
• /proc: es necesaria solo para que el kernel pueda correr diferentes procesos. Usualmente no se
modifica.
• /root: es el directorio del usuario root.
• /run: es una carpeta para configuración privada y configuraciones del usuario root.
• /sbin: similar a /bin, pero esta pensado para trabajar con el superusuario.
• /srv: contiene archivos de servicios instalados por el sistema.
• /tmp: donde se almacenan los archivos temporales del sistema.
• /usr: contiene todos los programas usados por un usuario regular.
• /var: contiene archivos que tienen un contenido variable, como archivos de registros y bases de
datos.
7) Comando listar: “ls”
• “ls -l” (muestra los archivos en detalle)
• “ls -a” (muestra los archivos ocultos y no ocultos)
• “ls -la” (combinación de los dos anteriores)

del comando que pusimos.
9) Los usuarios en Windows 10 se guardan en “C:\Users”
10) Los dos intérpretes de comandos más comunes en Windows son: “CMD” y “PowerShell”, siendo CMD

11) Directorios Windows 10:
• \Intel: directorio del procesador Intel.
• \PerfLogs: almacena los problemas del sistema y otros informes sobre el rendimiento.
• \ProgramFiles: hay carpetas de aplicaciones instaladas en el sistema operativo. Además, puede
haber carpetas en las que están configuraciones guardadas y otros datos / módulos comunes de
aplicaciones instaladas. "Common Files"
• \ProgramFiles (x86): se encuentran aplicaciones instaladas en Windows dedicados a la
arquitectura en 64-bits.
• \Users: se encuentra una carpeta por cada usuario, donde se almacena su información personal.
• \Windows: donde se guarda información y datos del sistema operativo.
12) SSH o Secure Shell, es un protocolo de administración remota que le permite a los usuarios
controlar y modificar sus servidores remotos a través de Internet a través de un mecanismo de
autenticación.
Proporciona un mecanismo para autenticar un usuario remoto, transferir entradas desde el cliente
